# Machine Learning Introduction: предсказание цен на аренду жилья

## О проекте

Вводный проект по машинному обучению, где я впервые построила модели для предсказания цен на аренду квартир. Проект охватывает полный цикл работы с данными: от первичного анализа до построения и сравнения моделей.


### Задача

Предсказать цену аренды квартиры на основе характеристик объявления:

* Кол-во ванных комнат (bathrooms)
* Кол-во спален (bedrooms)
* Уровень интереса арендаторов (interest_level)

Датасет: Kaggle Two Sigma Connect (Rental Listing Inquiries)
(https://www.kaggle.com/competitions/two-sigma-connect-rental-listing-inquiries/data)


### Что я делала

1. Первичный анализ данных
- Изучила структуру данных
- Определила целевую переменную (price)
- Проанализировала распределение цен, выявила выбросы
- Удалила экстремальные значения (1-й и 99-й процентили)

2. Статистический анализ
- Построила гистограммы распределения цен
- Создала boxplot для визуализации выбросов
- Закодировала категориальную переменную interest_level
- Проанализировала корреляции между признаками

3. Интересные моменты
- Сильная корреляция между ценой и кол-вом ванных (0.67)
- Средняя корреляция с кол-вом спален (0.55)
- Обратная зависимость: чем выше интерес арендаторов, тем ниже цена (-0.2)

4. Построение моделей
Сравнила 4 подхода: Decision Tree, Linear Regression, Naive (median), Naive (mean). Вычислила MAE и RMSE. Decision Tree показало лучшие результаты по обеим метрикам


### Технологии

* Python 3 + Jupyter Notebook
* Библиотеки: pandas, numpy, matplotlib, seaborn
* ML-библиотеки: scikit-learn (LinearRegression, DecisionTreeRegressor)
* Метрики: MAE (Mean Absolute Error), RMSE (Root Mean Squared Error)


### Чему научилась

- Работа с реальными данными Kaggle
- Предобработка данных (обработка выбросов, кодирование категорий)
- Построение и оценка ML-моделей
- Интерпретация метрик качества (MAE, RMSE)
- Визуализация данных (гистограммы, scatter plots, heatmaps)


### Некоторые мои выводы

* Относительно простые модели могут работать хорошо: Decision Tree превзошла линейную регрессию
* Качество данных критически важно: удаление выбросов сильно улучшило результаты
* Не все признаки одинаково полезны: кол-во ванных оказалось важнее спален
* Базовые модели имеют значение: наивные модели дали понимание минимального качества
